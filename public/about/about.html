About

This project aims to increase the accountability and accessibility of local governance. This site contains data from the Georgia Government Transparency and Campaign Finance Commission: for each sitting state senator, it scrapes all the itemized contributions that they have received during their time as a state senator. It calculates some associated stats, such as each senatorâ€™s top donors, and top donors to state senators overall. It originally began as a project for Codecademy Premium, which is an online coding bootcamp.

The web app consists of two parts. The backend consists of a node.js bot scraper that uses Headless Chrome/Puppeteer to scrape the GA State Senate site and the GA state media ethics site. It persists this data in Firebase Firestore and also Google Cloud Storage (for senator photos). The frontend is built with HTML, CSS, JS, and JQuery.

Currently, the bot scraper only works on the GA State Media Ethics sites, and is only configured to gather data on State Senators. If you would like to contribute to the project and extend it to other legislative bodies or states, or take it federal, all code can be found on github (node.js bot scraper repo; client repo). You will need to configure Firebase for yourself if you would like to contribute; or, if you shoot me an email I can give you write privileges for the Firebase project I am using.
